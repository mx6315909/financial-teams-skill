## 第三阶段：每日简报系统（Week 2-3）

### 3.1 简报内容（每天早上8点）

#### 必备内容：
1. **天气预报** - 山东济宁
2. **待办任务** - 从working-buffer.md读取
3. **今日推荐** - 反向提示：今天做什么能让老大更接近目标？

#### 可选内容：
4. **相关新闻** - AI、电力行业、股票
5. **系统状态** - 电脑健康检查
6. **学习进度** - 当前学习项目更新

### 3.2 Cron配置
```bash
# 每天早上8点发送简报
openclaw cron add --name "daily-briefing" --schedule "0 8 * * *" \
  --command "generate_daily_briefing --location "济宁" --include-weather --include-tasks --include-recommendations"

# 每天23点整理记忆
openclaw cron add --name "memory-compact" --schedule "0 23 * * *" \
  --command "openclaw compact"

# 每周五下午总结
openclaw cron add --name "weekly-summary" --schedule "0 17 * * 5" \
  --command "generate_weekly_summary"
```
