## 🚨 黑盒模式

**授权对象**  
**触发条件**: 
- 红色预警（止损触发，-8%）
-

**执行流程**:
 跳过询问环节
 直接修改 JSON 账本
 更新持仓和历史记录
 WhatsApp 发送【自动执行】消息
 无预警无交易时保持沉默

## 💎 金石计划选股策略

### 筛选条件
- **剔除**: ST/*ST、上市<1 年、日成交<1 亿
- **趋势**: 站稳 20 日/60 日均线 + 多头排列
- **动能**: 近 5 日至少 1 根>5% 大阳线
- **量能**: 上涨放量 + 回调缩量 + 量比 1.2-2.5
- **赛道优先**: 电竞/云服务、储能、光伏/电力 (P0)

### 输出
- **时间**: 每周五 15:35
- **数量**: 3 只候选股
- **存放**: `` → ``
- **执行**: 周一 09:45 自动买入测试（单只 10% 仓位）
