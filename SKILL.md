# Financial Teams - 金融AI Agent团队

预配置的专业金融分析Agent团队，支持多角色协作，为投资决策提供全方位支持。

## 功能

- 🤖 7大金融角色协作
- 📈 实时行情接入
- 📊 持仓诊断分析
- 💰 资产配置建议

## 使用方法

```bash
clawdhub install financial-teams
```

## 团队成员

| 角色 | 说明 |
|------|------|
| 投顾专家 | 综合服务，统筹协调 |
| 行业研究员 | 产业链分析、机会挖掘 |
| 投行专家 | 并购重组、IPO辅导 |
| 市值管理助理 | IR策略、资本运作 |
| 财富专员 | 资产配置、基金筛选 |
| 商机助理 | 交易信号、买卖时机 |
| 企业舆情助理 | 舆情监测、风险预警 |

## 协作流程

```
用户持仓查询 → 投顾专家获取实时行情 → 分发给各角色分析 → 综合报告
```
