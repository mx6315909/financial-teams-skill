## 🎯 核心概念（5个）

### 1. 实战案例复盘方法论
- **成功案例特征**：基本面优秀 + 技术面共振 + 估值合理 + 严格执行计划
- **失败案例教训**：追涨杀跌 + 仓位过重 + 没有止损 + 死扛到底
- **复盘四步法**：还原全过程 → 分析对错 → 总结教训 → 更新系统

### 2. 主力行为识别（从成交量看主力）
- **建仓期**：长期下跌后企稳 + 成交量间歇性放大 + 价格不创新低
- **拉升期**：突破关键位置 + 成交量持续放大 + 价格上涨迅速
- **出货期**：高位横盘不创新高 + 成交量持续放大但价格不涨 + 换手率>15%
- **洗盘期**：上涨趋势中突然下跌 + 很快企稳不跌破支撑 + 放量下跌快速缩量企稳

### 3. 交易系统五大模块
- **选股系统**：基本面初筛 → 技术面精选 → 估值筛选 → 热点催化
- **买入系统**：突破买入 + 回调买入 + 底部反转（3大信号）
- **卖出系统**：目标止盈 + 技术止盈 + 止损（坚决执行）
- **仓位管理系统**：分散≤20% + 动态调整 + 分批建仓 + 盈亏管理 + 保留现金
- **心态管理系统**：交易计划 + 交易日志 + 定期复盘 + 情绪管理 + 生活平衡

### 4. 交易检查清单（买入前必查）
- 选股检查：ROE>15%、MA多头、MACD零轴上、PE合理
- 买入检查：信号明确、成交量配合、大盘环境好
- 风险控制：止损位设定、仓位≤20%、分批建仓
- 心态检查：不追高、不重仓、不情绪化

### 5. 概率思维 vs 预测思维
- **以前**：炒股=预测涨跌，找到"必胜公式"
- **现在**：炒股=概率游戏，做对时多赚，错时少亏
- **核心**：不是"预测对"，而是"错了怎么办"（止损）

## ❓ 难点疑问

### 需要深入理解的内容：

1. **如何平衡"止损"和"被震荡洗出"？**
   - 可能需要结合技术位 + 固定比例，给一定缓冲空间
   - 或采用移动止损（随价格上涨上移止损位）

2. **如何判断是"正常回调"还是"趋势反转"？**
   - 需要结合成交量、基本面、大盘环境综合判断
   - 关键信号：是否跌破重要支撑 + 成交量是否异常

3. **如何避免"卖飞"（卖早了）？**
   - 分批止盈策略：达到目标卖出50%，剩余设移动止损
   - 让利润奔跑，但保护本金

4. **如何克服人性弱点（贪婪、恐惧、侥幸）？**
   - 形成固定交易系统，用纪律约束情绪
   - 写交易计划，严格执行，定期复盘

5. **周期股如何判断周期位置？**
   - 需要学习行业周期规律（供需/库存/价格）
   - 周期股低PE时警惕（周期顶部），高PE时关注（周期底部）

## 💡 今日核心感悟

### 1. 系统比单点重要
不是找一个"必胜指标"，而是建立完整闭环：选股 → 买入 → 卖出 → 仓位 → 风控

### 2. 纪律比聪明重要
简单系统 + 严格执行 > 复杂系统 + 随意执行

### 3. 风控比收益重要
先想怎么不亏钱，再想怎么赚钱。亏损50%需要涨100%才能回本。

### 4. 概率比预测重要
追求长期正期望，不追求单次正确。关键是"错了怎么办"。

### 5. 实战案例的价值
从茅台中石油的案例中，真正理解了"再好的公司价格过高也不能买"和"周期股陷阱"。
