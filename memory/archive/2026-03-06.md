### 1. proactive-agent-skill (🚀)
**功能**：将AI代理从任务跟随者转变为主动合作伙伴
**核心特性**：
- WAL协议（预写日志）- 保护关键状态，防止上下文丢失
- Working Buffer - 危险区域日志，防止会话重启丢失关键信息
- 自主Cron任务 - 定时自动化任务
- Heartbeat检查 - 定期检查邮件、日历、天气等

**文件结构**：
```
workspace/
├── MEMORY.md              # 长期策划记忆
├── memory/YYYY-MM-DD.md   # 每日原始日志
├── SESSION-STATE.md       # 活跃工作记忆
└── working-buffer.md      # 危险区域日志
```

### 2. summarize-pro (📝)
**功能**：专业文本总结引擎，20种总结格式
**核心特性**：
- 本地处理，无外部API调用，数据完全本地
- 支持多种格式：TL;DR、要点、ELI5、执行摘要、会议记录等
- 多语言支持（包括中文）
- 自定义长度总结
- 历史记录和统计

**常用命令**：
- `summarize [text]` - 默认总结
- `tldr [text]` - 1-2句快速总结
- `bullets [text]` - 要点形式
- `eli5 [text]` - 简单解释
- `key takeaways [text]` - 关键要点
- `action items [text]` - 提取待办事项
- `meeting summary [text]` - 会议记录格式
- `summarize in chinese [text]` - 中文总结

**数据存储**：`~/.openclaw/summarize-pro/`
