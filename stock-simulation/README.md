# 📈 7 天股票模拟操作挑战

## 🎯 挑战目标

使用 **Financial Teams 7 大专员**协作管理 10 万元虚拟资金，进行 7 天股票模拟操作，目标是**跑赢大盘（沪深 300）**。

---

## 🤖 专家团队

| 角色 | 职责 |
|------|------|
| 📊 **投顾专家** | 统筹决策、最终审核 |
| 🔬 **行业研究员** | 选股分析、行业趋势 |
| 💼 **投行专家** | 估值分析、基本面 |
| 📈 **市值管理助理** | 资本运作、消息面 |
| 💰 **财富专员** | 仓位管理、资产配置 |
| 🎯 **商机助理** | 买卖时机、技术面 |
| 🚨 **企业舆情助理** | 风险排查、舆情监控 |

---

## 📅 每日流程

| 时间 | 任务 | Cron ID |
|------|------|---------|
| **09:00** | 早盘分析（选股/持仓诊断） | `0f65d487-f672-463e-8c50-cee601f75f1c` |
| **15:30** | 收盘总结（当日操作记录） | `ca5e46fa-c8d9-4575-8f8c-b8d61be8f039` |
| **23:00** | 深度复盘（学习总结） | `a51f74b1-ea71-46dd-a9db-0a356a70c40a` |

---

## 📊 交易规则

| 规则 | 限制 |
|------|------|
| **初始资金** | ¥100,000 |
| **最大仓位** | 单只股票≤30% |
| **最大持股数** | 5 只 |
| **止损线** | -8% |
| **止盈线** | +20% |
| **交易时间** | 09:30-11:30, 13:00-15:00 |
| **市场** | A 股（沪深） |
| **买入时机** | 开盘后 2-3 分钟（09:32-09:33） |
| **交易费用** | 佣金 0.025% + 印花税 0.05%（卖出）+ 过户费 0.002% |
